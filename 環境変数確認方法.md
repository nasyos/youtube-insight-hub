# 環境変数確認方法

## 🔍 環境変数が正しく読み込まれているか確認

### 方法1: ブラウザのコンソールで確認

1. ブラウザで `http://localhost:3000`（または `http://localhost:3002`）を開く
2. 開発者ツール（F12）を開く
3. Consoleタブを確認

**正常な場合**:
```
🔍 環境変数チェック:
  VITE_SUPABASE_URL: https://vgscuhunmcpozsjlorfh.supabase.co...
  VITE_SUPABASE_ANON_KEY: sb_publishable_HsmW3Hd0kQ...
✅ Supabase接続設定が確認されました

🔍 Gemini APIキーチェック: AIzaSyC0fIP7Nq1PdM4ttgw...
✅ Gemini APIキーが正常に設定されました
```

**エラーの場合**:
```
⚠️ Supabase環境変数が設定されていません...
⚠️ Gemini APIキーが設定されていません...
```

### 方法2: .env.localファイルの確認

`.env.local` ファイルが以下の形式になっているか確認：

```env
VITE_SUPABASE_URL=https://vgscuhunmcpozsjlorfh.supabase.co
VITE_SUPABASE_ANON_KEY=sb_publishable_HsmW3Hd0kQ4oagj14fm-9A_MIx9Q30r
VITE_GEMINI_API_KEY=AIzaSyC0fIP7Nq1PdM4ttgwi004LB9QHV
```

**重要なポイント**:
- ✅ `VITE_` プレフィックスが必要
- ✅ 値に引用符（`"` や `'`）は不要
- ✅ 値の前後に余分なスペースがないか確認
- ✅ 各環境変数は1行に1つ

### 方法3: 開発サーバーの再起動

環境変数を変更したら、**必ず**開発サーバーを再起動：

1. ターミナルで `Ctrl + C` でサーバーを停止
2. `npm run dev` で再起動
3. ブラウザをリロード（F5）

## ⚠️ よくある問題

### 問題1: 環境変数が読み込まれない

**原因**:
- 開発サーバーを再起動していない
- `.env.local` ファイルの形式が間違っている
- ファイルが正しい場所にない（`package.json`と同じディレクトリ）

**解決方法**:
1. `.env.local` ファイルを確認
2. 開発サーバーを再起動
3. ブラウザをリロード

### 問題2: Supabase接続エラー

**症状**: `Supabase環境変数が設定されていません` と表示される

**確認項目**:
1. `VITE_SUPABASE_URL` が正しく設定されているか
2. `VITE_SUPABASE_ANON_KEY` が正しく設定されているか
3. 値に余分なスペースや引用符がないか

**例（正しい形式）**:
```env
VITE_SUPABASE_URL=https://vgscuhunmcpozsjlorfh.supabase.co
VITE_SUPABASE_ANON_KEY=sb_publishable_HsmW3Hd0kQ4oagj14fm-9A_MIx9Q30r
```

**例（間違った形式）**:
```env
VITE_SUPABASE_URL="https://vgscuhunmcpozsjlorfh.supabase.co"  # ❌ 引用符は不要
VITE_SUPABASE_URL = https://vgscuhunmcpozsjlorfh.supabase.co  # ❌ 等号の前後にスペースは不要
```

### 問題3: Gemini APIキーが読み込まれない

**症状**: `Gemini APIキーが設定されていません` と表示される

**確認項目**:
1. `VITE_GEMINI_API_KEY` が正しく設定されているか
2. 値が `your_gemini_api_key_here` になっていないか（プレースホルダーのまま）
3. APIキーが正しい形式か（通常は `AIza...` で始まる）

## 📝 確認手順まとめ

1. ✅ `.env.local` ファイルを確認
2. ✅ 開発サーバーを再起動（`Ctrl+C` → `npm run dev`）
3. ✅ ブラウザをリロード（F5）
4. ✅ コンソール（F12）でデバッグメッセージを確認
5. ✅ `✅ Supabase接続設定が確認されました` が表示されるか確認
6. ✅ `✅ Gemini APIキーが正常に設定されました` が表示されるか確認
